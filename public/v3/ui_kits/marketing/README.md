# Marketing UI Kit

High-fidelity recreation of the SolSentry marketing surface — the landing page and social lockups.

Informed by:
- `assets/twitter-banner-1500x500.jpg` (PT-BR hero, four pillar buttons, metrics row)
- `assets/banner-wordmark.png` (wordmark lockup with eyebrow + tagline)
- Docs README (metrics, case study, competitive table)

## Components
- `MarketingNav.jsx` — top nav with logo + anchors
- `Hero.jsx` — SOLSENTRY display wordmark + tagline + pillar chips + metrics row
- `PillarGrid.jsx` — four-up pillar cards
- `CaseStudyBlock.jsx` — 4kxscute operator case study
- `CompetitiveTable.jsx` — capability matrix vs competitors
- `FooterMark.jsx` — wordmark footer
